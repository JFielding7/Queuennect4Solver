# Queuennect 4 Solver

A full-stack application with Rust backend and a React frontend

## Features

* **Interactive Sandbox:** Use the engine to evaluate any position
* **Deep Game Analysis:** Review the game history
* **Branching Timelines:** Click the board during an analysis to explore alternative lines.
* **Engine:** Built in Rust uses multiple threads for performance

## Prerequisites

* [Node.js](https://nodejs.org/)
* [Rust & Cargo](https://www.rust-lang.org/tools/install)

## Installation & Setup

### Start the Rust Backend

```bash
cd crates/server

cargo run
```

### Start the React Frontend

```bash
cd frontend

npm i

npm run dev
```
